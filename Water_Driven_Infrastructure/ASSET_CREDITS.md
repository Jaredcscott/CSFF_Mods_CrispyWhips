# Asset Credits

This repository contains custom card artwork and working art files for Card Survival: Fantasy Forest mods.

## Current Shipped Mod Art

The shipped PNG assets under each mod's `Resource/Picture/` directory are maintained as project assets for these mods. During the April 26, 2026 compliance pass, no attribution-required third-party image source was identified in the shipped `Resource/Picture/` assets.

## Working Files

The `Assets/` directory contains editable source art files and experiments. Files with names that indicate an external generation service, such as `openart-image_*.xcf`, should be treated as generated working material and reviewed against the service's current terms before being shipped in a public release.

## Future Additions

When adding external, commissioned, AI-generated, or derived artwork, record the source, author/tool, license or terms, and any attribution text here before publishing the mod package.
