# Skill Speed Boost - Future Feature Ideas

## Core System Expansion

### Enhanced Staleness Mechanics
- **Granular Skill Decay** — Different decay rates for different skills (crafts decay faster than survival skills)
- **Partial Credit System** — Each action gives credit toward decay, not all-or-nothing
- **Combo Streaks** — Bonus XP for performing related skills in sequence
- **Cross-Skill Synergy** — Performing skill A gives partial decay prevention for skill B
- **Specialty Decay** — Some skills decay slower/faster based on player level

### Advanced XP Multipliers
- **Difficulty Multipliers** — Harder actions grant more XP
- **Level-Based Scaling** — XP multiplier increases with skill level
- **Dynamic Multipliers** — Multiplier changes based on game progression
- **Tool Quality Bonuses** — Better tools grant higher XP multipliers
- **Stat Bonuses** — Physical stats affect XP gains
- **Gear Quality Effects** — Armor/equipment affects learning rate

## Skill Progression Systems

### Prestige/Mastery System
- **Skill Prestige** — Reset skill to gain "mastery points"
- **Mastery Abilities** — Unlock special perks at high mastery
- **Skill Specialization** — Choose skill focus for bonuses
- **Expert Mode** — Harder challenges with better XP rewards
- **Master Perks** — Unique abilities unlocked at skill cap

### Skill Trees
- **Branching Skill Paths** — Each skill has multiple advancement options
- **Talent Points** — Spend at level-up for skill bonuses
- **Passive Bonuses** — Persistent XP/speed multipliers from perks
- **Active Abilities** — Special moves unlocked from skills
- **Hybrid Trees** — Skills that combine multiple capabilities

## Gameplay Mechanics

### Skill Synergies
- **Related Skill Bonuses** — Smithing + Tool Use give combined bonuses
- **Equipment Synergies** — Certain gear combos boost specific skills
- **Stat-Skill Connections** — High strength helps smithing
- **Environmental Bonuses** — Forage bonus in forests, fishing bonus at rivers
- **Companion Bonuses** — Skill boosts from party members/perks

### Skill Challenges
- **Skill Quests** — Complete specific challenges for XP bursts
- **Timed Challenges** — Speed challenges for bonus XP
- **Difficulty Tiers** — Progressive challenge difficulty
- **Skill-Specific Events** — Random events that boost learning

## Leveling Customization

### User-Defined Multipliers
- **Per-Skill Multipliers** — Set different multiplier for each skill
- **Time-Based Multipliers** — Higher multiplier for off-peak grinding
- **Progress-Based Scaling** — Multiplier changes as you level
- **Activity-Based Costs** — Cost-benefit system for grinding
- **Balance Presets** — Pre-configured multiplier sets (casual, hardcore, etc.)

## Implementation Priority

**High Priority:**
- Per-skill multiplier customization (frequently requested)
- Skill synergies (adds depth, minimal complexity)
- Better UI indicators (quality of life essential)
- Prestige/mastery basic system (long-term content)

**Medium Priority:**
- Skill trees (major feature, design-heavy)
- Difficulty modes (presets for different playstyles)
- Advanced XP scaling (complex formulas)
- Session tracking/analytics
- Character progression specialization

**Low Priority (High effort):**
- Dynamic skill economy (requires balance tuning)
- Complex skill interactions
- Machine learning recommendations
- Full social/community features
- Cross-save progression system

## Known Challenges
- Balancing to not make vanilla feel wrong
- Preventing XP farming exploits
- Ensuring all skills remain useful
- Performance impact of complex calculations
- Player preference diversity
